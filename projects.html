Projects

I have worked with many different tools and frameworks throughout my experience with App Academy. Below are Listed some of the different applications that I have build by myself or with a group of my peers.

Raynes Closet

This application is an e-commerce site I had designed for my younger sister to assist her business in boosting revenue.

Main Features include:
• A fully Functional chart
• The Ability to have distinguished roles.
• The Ability to view/edit orders
• The Ability for a Master Account to grant or Remove Access for an account to become an Admin Account.
• The Ability to Add/edit products as well as update inventory.
• The Ability for consumers to place suggestions for products that they would like to see.

    Tools and FrameWorks Used:
  • - React
  • - Flask
  • - Node JS
  • - Express
  • - JavaScript

Yum Cart

This application is a clone of the Popular site UberEats!

Main Features include:
• A fully Functional chart
• The Ability create/edit/delete Restaurants.
• The Ability to create/edit/delete Menus for respective Restaurants
• The Ability to create/edit/delete Menu Items for respective Menus

Tools and FrameWorks Used:

  • - React
  • - Flask
  • - Node JS
  • - Python
  • - JavaScript
  • - Docker
